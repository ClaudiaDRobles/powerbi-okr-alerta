# 📊 Automatización de Reportes OKR (POC)

## 🚀 Descripción

Este proyecto es una prueba de concepto (POC) que demuestra la integración entre Power BI y Power Automate para automatizar el envío de reportes de cumplimiento de objetivos (OKR).

El objetivo es simular un escenario real de negocio donde los stakeholders reciben reportes ejecutivos con un solo clic.

---

## ⚙️ Tecnologías

* Power BI
* DAX
* Power Automate
* HTML (correo dinámico)

---

## 📈 Funcionalidad

* Visualización de métricas por dominio
* Uso de medidas DAX para cálculo de indicadores
* Botón en Power BI que dispara flujo automático
* Generación de contenido HTML dinámico
* Envío de correo con resultados

---

## 🔄 Flujo

1. Usuario interactúa con el dashboard
2. Se envían datos a Power Automate
3. Se transforman en formato HTML
4. Se envía un correo automatizado

---

## 💡 Caso de uso

Este tipo de solución se utiliza en:

* Seguimiento de OKR
* Reportes de cumplimiento
* Monitoreo de indicadores
* Automatización de reporting ejecutivo

---

## ⚠️ Nota

Los datos utilizados en este proyecto son simulados con fines demostrativos.
